## Files

| --------------------- | ----------------------------------- | ----------------------------------------- |
| `config.json`         | User configuration (paths, git)     | Configuring plugin for your environment   |
| `example-config.json` | Default configuration template      | Setting up new installation               |
| `topic-patterns.json` | Topic detection glob patterns       | Adding topics, modifying detection rules  |
